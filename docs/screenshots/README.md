# Иллюстрации интерфейса 

Сюда помещайте скриншоты работающего приложения для включения в печатную версию руководства. Рекомендуемые файлы и содержание:

| Файл | Что снять |
|------|-----------|
| `01-guest-catalog.png` | Окно: боковое меню, кнопка «Каталог услуг», список услуг |
| `02-login.png` | Форма авторизации |
| `03-register.png` | Форма регистрации |
| `04-services-moderator.png` | Каталог с кнопками «Добавить» / «Ред.» (вход модератора) |
| `05-profile.png` | «Мой профиль»: баланс, пополнение, запись, отзыв |
| `06-moderator.png` | Панель модератора |
| `07-master.png` | Панель мастера |
| `08-admin.png` | Панель администратора |

В `USER_GUIDE.md` используются также сгенерированные схематичные иллюстрации (`illustration-*.png`) в этой же папке.
